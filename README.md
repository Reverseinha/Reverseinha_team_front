# 🪄 Code Convention

- **변수 작성 방법**
  - 컴포넌트 이름, 파일명: 파스칼 케이스 (ex. MyVariable)
  - 함수 및 변수, 핸들러 이름: 카멜 케이스 (ex. getUserName)

- **폰트 적용 방식**
  - font-face
    - Pretendard-Regular

- **요소 크기 지정 방식**
  - 단위 rem
 
- **컬러 팔레트**
  - #004EE5
  - #A2A2A2
